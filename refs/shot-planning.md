# 镜头规划指南

---

## 第一节：剧本格式

在规划任何镜头之前，必须先完成本集剧本。

### 剧本输出格式

```
=== 《剧名》第X集 剧本 ===
主题：[本集核心情感/主题]
结构：[本集在整体故事中的位置，如"第一集·相遇"]

【第一场】
地点：[具体场景]  时间：[时间段]
（场景描述：环境气氛、人物初始状态）

[角色A]：台词。（情绪/肢体动作备注）
[角色B]：台词。（情绪备注）
（内心独白或旁白另起一行，用括号）

本场结尾钩子：[说明如何为下一场制造悬念或情绪拉力]

【第二场】
...
```

### 台词质量要求

| 问题 | 正确做法 |
|------|----------|
| ❌ 信息堆砌："你是宫里来的人，你掌管东厂，你要抓我对吗？" | ✅ 留白克制："你来了。" / "该来的总会来。" |
| ❌ 没有动机："我很伤心。" | ✅ 行为化情绪："（转过身，背对他）没什么。" |
| ❌ 对白互相重复 | ✅ 对白形成张力：A 说一件事，B 回应另一件事 |
| ❌ 每个镜头都有台词 | ✅ 沉默镜头与台词镜头交替，给情绪呼吸空间 |

## 第二节：镜头规划格式

剧本确认后，逐场分配镜头。

### 标准单集结构（6-8 个镜头为佳）

```
镜头 1：开场建立（大远景/远景）
  └─ 交代时间、地点、基础氛围
  └─ 通常无台词，纯画面

镜头 2-3：人物引入（全景/中景）
  └─ 主角出场，确立人物状态
  └─ 可有第一句台词

镜头 4-5：冲突发展（中景/近景交替）
  └─ 核心对话或动作推进
  └─ 景别交替：中景→近景→中景，避免连续同景别

镜头 6-7：情感高潮（近景/特写）
  └─ 情绪最强点，关键台词或关键动作
  └─ 此处 BGM 可强化或特殊处理

镜头 8：结尾（中景或远景）
  └─ 情绪余波 + 悬念钩子
  └─ 通常无台词，用画面说话
```

### 镜头规划表完整格式

```
=== 《剧名》第X集 镜头规划 ===

美术风格：[全集统一描述]
色调基准：[全集基础色调]

────────────────────────────
【镜头 1】[开场]
所属场：第一场
前接：— （集首）
后接：镜头2 起始状态（例：切入走廊全景）
过渡方式：淡入
景别：大远景
角度：俯角
主体：空镜，黄昏时分的古城楼
背景：夕阳西下，城楼轮廓，飞鸟掠过
氛围：橙金色调，史诗感，静谧
台词：无台词
动态：云彩缓缓移动，飞鸟轻扫过画面
音频：[直接填写音频设计单内容，例：古风琵琶BGM，远处城市喧嚣声]

────────────────────────────
【镜头 2】[人物引入]
所属场：第一场
前接：镜头1 结束（鸟飞过城楼后）
后接：镜头3 起始（女主转身面向镜头）
过渡方式：硬切
景别：全景
角度：平视
主体：女主角独立站于城楼边缘，背对镜头
背景：同镜头1的城楼，背景保持一致
氛围：逆光，发丝飘动
台词：无台词（沉默镜头）
动态：发丝随风缓缓飘动，衣袂轻轻拂动
音频：[同上]

────────────────────────────
【镜头 3】[...继续]
```

## 第三节：景别与角度参考

### 景别体系

| 景别 | 画面范围 | 典型用途 |
|------|----------|----------|
| 大远景 | 宏大环境，角色不可见或极小 | 开场建立、史诗感、时间跳跃 |
| 远景 | 全身 + 大量背景 | 交代地点、人物出场 |
| 全景 | 全身，背景可见 | 动作展示、人物关系 |
| 中景 | 腰部以上 | 对话、情感交流 |
| 近景 | 胸部以上 | 情绪特写、重要台词 |
| 特写 | 面部/手部/道具 | 关键细节、情感爆发点 |

**景别节奏规则：**
- 相邻两个镜头不用相同景别（会造成视觉跳跃感）
- 对话场景：近景↔中景 交替，营造呼吸感
- 情绪推进：景别逐渐缩小（远→中→近→特写）= 情绪升级

### 镜头角度

| 角度 | 效果 |
|------|------|
| 平视 | 客观叙事，最常用 |
| 仰角（低机位） | 主角威严、英雄感 |
| 俯角（高机位） | 角色渺小、孤独、被压制 |
| 斜侧面 | 戏剧张力，对峙/冲突 |
| 背面 | 神秘感，引导观众视角 |

## 第四节：衔接规则（必须遵守）

### 空间连续性
- **同一场景内**：所有镜头的背景描述使用相同的核心词汇
  - ✅ 正确：镜头3背景"宫廷走廊，红柱灯笼"，镜头4背景"同一走廊，红柱灯笼，景深更近"
  - ❌ 错误：镜头3"宫廷走廊"，镜头4突然变成"御花园"（没有过渡镜头）

### 动作连续性
- 前一镜头"后接"字段与下一镜头"前接"字段必须逻辑衔接
  - ✅ 镜头2后接："人物转头"→ 镜头3前接："人物面向镜头"
  - ❌ 镜头2后接："人物背身"→ 镜头3前接："人物已走到屋内"（跳跃）

### 过渡方式选择
| 过渡 | 适用场景 |
|------|----------|
| **硬切** | 节奏快、对话场景、同一空间内切换 |
| **淡入淡出** | 时间跳跃、场景切换、情绪沉淀 |
| **推近** | 情绪升级、悬念揭示 |
| **拉远** | 情绪收尾、孤独感、史诗结尾 |

### 台词连贯性检查
规划完成后，按顺序读出所有"有台词"镜头的对话，验证：
- 逻辑是否通顺？
- 有没有信息重复？
- 情绪节奏是否有起伏（不能平铺直叙）？

## 第五节：不同类型漫剧风格参考

| 类型 | 景别偏好 | 色调 | BGM 参考 |
|------|----------|------|----------|
| 热血/战斗 | 仰角近景、特写 | 高对比橙红 | 激昂电子摇滚、鼓点强烈 |
| 古风/言情 | 平视中景、柔和近景 | 低饱和暖黄青绿 | 古风琵琶、古筝、箫声 |
| 都市/现代 | 斜侧面、低角度 | 霓虹冷蓝/暖橙 | 都市电子、钢琴爵士 |
| 悬疑/惊悚 | 俯角、特写 | 冷灰深蓝高对比 | 弦乐紧张、间歇静默 |
| 温情/治愈 | 平视中景、远景 | 低饱和暖色 | 轻钢琴、吉他 |

## 第六节：角色一致性要点

每个镜头的分镜图提示词必须包含角色核心外貌，跨镜头不得改变：

**必须固定的要素：**
- 发型（颜色 + 样式，如"银色长发双马尾"）
- 眼睛颜色（如"紫色眼眸"）
- 核心服装（如"黑色校服，红色蝴蝶结"）
- 显著特征（如"左脸颊有星形刺青"、"始终持一把折扇"）

**第一个镜头**写出完整描述，后续镜头必须逐字复用同样的外貌描述串。
