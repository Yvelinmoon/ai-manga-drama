# 镜头规划指南

适用于 `ai-manga-drama` 技能的 Step 2：镜头规划。

---

## 一、景别体系

| 景别 | 英文 | 画面范围 | 典型用途 |
|------|------|----------|----------|
| 大远景 | Extreme Wide Shot | 宏大环境，角色极小或不可见 | 开场世界观建立、史诗感 |
| 远景 | Wide Shot | 全身 + 大量背景 | 交代地点、出场亮相 |
| 全景 | Full Shot | 全身，背景可见 | 动作展示、人物关系 |
| 中景 | Medium Shot | 腰部以上 | 对话、情感交流 |
| 近景 | Close-up | 胸部以上 | 情绪特写、重要台词 |
| 特写 | Extreme Close-up | 面部/手部/道具 | 关键细节、情感爆发 |

---

## 二、镜头角度

| 角度 | 效果 |
|------|------|
| **平视** | 客观叙事，最常用 |
| **仰角（低机位）** | 主角显得高大、威严、英雄感 |
| **俯角（高机位）** | 角色显得渺小、脆弱、孤独 |
| **斜侧面** | 增加戏剧张力，适合对峙/冲突 |
| **背面** | 神秘感，引导观众视角 |

---

## 三、漫剧节奏规划

### 标准单集结构（6-12 个镜头）

```
镜头 1-2：开场建立
  └─ 大远景/远景 → 交代时间、地点、氛围

镜头 3-4：人物引入
  └─ 全景/中景 → 主角出场，建立人物状态

镜头 5-7：冲突发展
  └─ 中景/近景交替 → 对话、矛盾推进

镜头 8-9：情感高潮
  └─ 近景/特写 → 情绪最强点，关键台词/动作

镜头 10-11：转折/悬念
  └─ 中景 + 特写 → 剧情转折，留下钩子

镜头 12：结尾留白（可选）
  └─ 远景/大远景 → 情绪沉淀，预告下集
```

## 四、不同类型漫剧的镜头风格

### 热血/战斗类
- 多用**仰角近景**和**特写**
- 大量动态镜头：拳击、爆炸、冲锋
- 节奏快，镜头切换频繁（6-8 个镜头即可）
- 色调：高对比、暖色调、橙红色

### 古风/言情类
- 多用**平视中景**和**柔和近景**
- 动态轻柔：衣袂飘动、眼神交流、发丝飞扬
- 节奏慢，镜头停留更长
- 色调：低饱和、暖黄、青绿、水墨感

### 都市/现代类
- 多用**斜侧面**和**低角度**
- 场景多样：写字楼、咖啡馆、夜晚街道
- 色调：高饱和、霓虹、冷蓝/暖橙对比

### 悬疑/惊悚类
- 多用**俯角**和**特写**
- 强调阴影、光影对比
- 慢镜头感：颤抖、慢转头、慢眨眼
- 色调：冷灰、深蓝、高对比黑白

### 喜剧类
- 多用**全景**展示身体语言
- 夸张的表情特写
- 明亮色调、高饱和度
- 适当使用对称构图

## 五、角色一致性要点

每个镜头的提示词必须包含角色核心外貌描述，确保跨镜头一致性：

**必须重复的要素：**
- 发型（颜色 + 样式）
- 眼睛颜色
- 核心服装描述
- 显著特征（疤痕、配饰、武器等）

**示例模板：**
```
[主角一致性描述] 一位18岁少女，银色长发双马尾，紫色眼眸，
穿着黑色校服，胸前别着红色蝴蝶结，
[本镜头特有描述] 站在屋顶上俯视城市，神情坚定，
[背景环境] 夕阳下的都市全景，橙色天空，
[氛围] 电影感光影，史诗氛围，高质量动漫风格
```

## 六、镜头规划输出模板

```
=== 《剧目名称》第 X 集镜头规划 ===

剧情摘要：[本集核心情节一句话]
美术风格：[风格描述]
情感基调：[基调]
镜头总数：N

────────────────────────────
【镜头 1】[开场/建立]
景别：大远景
角度：平视/俯角
主体：空镜，城市夜景
背景：霓虹闪烁的都市，高楼林立
氛围：冷蓝色调，孤独感，夜晚
动态：城市灯光缓缓闪烁，云层轻缓移动

【镜头 2】[主角出场]
景别：全景
角度：仰角
主体：女主角站在屋顶边缘，银发随风飘动
背景：夜空+城市远景
氛围：逆光，轮廓光勾勒，神秘
动态：发丝和外套衣角随风缓缓飘动，她缓缓转头

...（继续到镜头 N）
────────────────────────────

请确认镜头规划，确认后开始生成分镜图。
```
