# 提示词模板

## 分镜图（make-image）

```
[角色核心外貌：发型/眼睛/服装] + [景别] + [姿势/表情] + [背景场景] + [光影氛围] + [艺术风格]
```

**景别关键词：**
- 大远景：`establishing shot, vast landscape`
- 全景：`full body shot`
- 中景：`medium shot waist up`
- 近景：`close-up portrait, bokeh background`
- 特写：`extreme close-up of [部位]`

**注意：** 每个镜头必须重复角色核心外貌描述，确保跨镜头一致性。

---

## 视频动态（make-video）

```
[主体 + 身份细节] 在 [环境 / 场景] 中。
主要动作：[使用具体动词的核心动作]。
次要动作：[可选的辅助动作]。
镜头：[运镜类型 + 速度]。
风格：[光线、氛围]。
节奏：[慢 / 中 / 快]。
声音：[环境音 / 人物台词 / 背景音乐]。
约束：保持 [特定元素] 稳定，避免 [常见瑕疵]。
```

**注意：** 只描述动态和音效，不重复图片中已有的静态内容。
